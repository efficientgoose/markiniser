# Phase 3 Web Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build the phase 3 React frontend shell for Markiniser with file tree browsing, debounced search, WebSocket tree updates, and a placeholder editor/preview layout.

**Architecture:** Replace the placeholder `packages/web` package with a Vite React app using Zustand for global state, a small API layer for server calls, and a WebSocket hook for live tree updates. Keep the shell editor-native and phase-3-scoped: file tree, search swap mode, layout chrome, resizable panels, placeholder main/preview panels, and `dirtyContent` in the store for future editor work.

**Tech Stack:** Vite, React, TypeScript, Tailwind CSS, Zustand, react-arborist, Vitest, React Testing Library
